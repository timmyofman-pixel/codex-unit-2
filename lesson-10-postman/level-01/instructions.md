## Level 01 — Install Postman

## Objective

- Install Postman and verify it launches.

## Benefits

- Quick verification that students can run a desktop/web tool used for APIs.
- Ensures consistent environment for later levels.

## Complete these tasks

- Download Postman from the official site and open it.
- Confirm the main workspace is visible.
- `console.log` the system (Windows, Mac, or Linux) you installed Postman for.
- `console.log` if the installation was easy or had difficulties.

## Hints

- Use the official downloads page at https://www.postman.com/downloads/.
- If the app doesn't open, try restarting your machine or checking permissions.

## More information

- Postman is a GUI client for sending HTTP requests and inspecting responses.
- Further reading: Postman docs (https://www.postman.com/docs/).

## Usage tips

- Keep Postman open while completing later levels.
