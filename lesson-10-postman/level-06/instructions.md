## Level 06 — API GET practice

## Objective

- Choose an API, read the docs, and implement a GET request demonstrating parameter and response handling.

## Benefits

- Reinforce independent API exploration and applying learned skills.
- Practice parsing and logging JSON responses in code.

## Complete these tasks

- Pick an API from the instructor list and find a suitable GET endpoint.
- Make the GET request and capture/inspect the response.
- Console.log a documentation detail and the parsed JSON in `script.js`.

## Hints

- Read the endpoint's example responses in the docs to know what fields to parse.

## More information

- Most APIs return JSON; use `response.json()` in browser code or Postman's viewer to inspect parsed objects.

## Usage tips

- Test the endpoint in Postman first, then replicate the request in `script.js`.
